# Code detect
## 介绍：
我的目标是涉及一款代码检测的工具，目前出现的代码检测工具例如MOSS，但是由于其繁琐的下载步骤以及部署条件，所以我希望在MOSS等工具的基础上进行优化和升级，并且解决这些繁琐的步骤，让查重的变得简单快捷，节省大量的时间。

## 与 Moss 比较：
1.）仅包含一个源文件

2.）无第三方依赖（使用 TreeDiff 时 zss 除外）

3.）无需注册帐户

4.）无需网络即可访问
